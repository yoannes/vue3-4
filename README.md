# vue3-4

## Fluxo de trabalho

1. Abre o github desktop
1. Criar um branch novo
   1. feature/xxx
   1. fix/xxx
1. Fazer o serviço
1. Adicionar os arquivos
1. Criar um commit
   1. Colocar mensagem (obrigatorio)
   1. Colocar um descrição
1. Push pro github
1. Cria um PR (pull request)
1. Espera validação
